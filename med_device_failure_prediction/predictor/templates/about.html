Project Overview

The Medical Device Failure Prediction system is built to support healthcare facilities in proactively identifying medical devices that are at risk of failure. Using machine learning techniques, the system predicts a device's failure class based on operational and maintenance data.

Input Features

The system takes the following inputs related to a medical device's operational and maintenance history:

  • Age of the Device
  • Maintenance Cost
  • Downtime
  • Maintenance Frequency
  • Failure Event Count

Output

The model predicts one of the following failure risk categories:

  • Class 0: No imminent failure expected
  • Class 1: Unlikely to fail within the first 3 years
  • Class 2: Likely to fail within 3 years
  • Class 3: Likely to fail after 3 years

Algorithms Used

Preprocessing Techniques:

  • Z-score normalization using StandardScaler
  • Cost per Event = Maintenance Cost / (Failure Events + 1)
  • Downtime per Frequency = Downtime / (Maintenance Frequency + 1)

Machine Learning Models:

  • Random Forest
  • Gradient Boosting (GridSearchCV tuned)
  • Support Vector Classifier (RBF)
  • Decision Tree
  • NaÃ¯ve Bayes

These are combined using a Weighted Voting Ensemble for optimal performance.
